# 🌈 Rainbow Picker

A modern and interactive React.js Color Picker application that allows users to explore the colors of the rainbow through a beautiful glassmorphism-inspired interface.

Built with **React + Vite**, this project demonstrates modern frontend development concepts including dynamic styling, state management, responsive design, animations, and user interaction.

---

## ✨ Features

### 🎨 Rainbow Color Selection

* Red
* Orange
* Yellow
* Green
* Blue
* Indigo
* Violet
* Rainbow Mode

### 🌈 Dynamic Background Changes

* Instantly updates the background when a color is selected.
* Smooth transitions between colors.
* Animated rainbow gradient mode.

### 💎 Modern UI

* Glassmorphism card design.
* Soft shadows and rounded corners.
* Fully centered responsive layout.
* Modern typography and spacing.

### ⚡ Interactive Experience

* Hover animations.
* Active click effects.
* Smooth color transitions.
* Responsive button interactions.

### 📱 Responsive Design

* Desktop friendly.
* Tablet optimized.
* Mobile responsive.
* Full-screen layout without unwanted margins.

### 📋 Additional Functionality

* Selected color display.
* HEX code display.
* Copy HEX code to clipboard.
* Recently selected color history.
* Keyboard navigation support.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/thanmai09/-Rainbow-Picker.git
```

Navigate into the project:

```bash
cd -Rainbow-Picker
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
